# BacktotheDate Mao-era poster sources

- For the BacktotheDate Chinese poster plugin, default image selection should target Great Leap Forward and Cultural Revolution era posters instead of the full chineseposters.net archive.
- Use chineseposters.net theme pages such as `great-leap-forward`, `cultural-revolution-campaigns`, and related Mao-era Cultural Revolution subthemes as the preferred candidate pool.
- The Shanghai People's Commune theme slug is `shanghai-commune`; `shanghai-peoples-commune` returns 404.
- Keep the full archive only as an explicit fallback/source mode, because it can surface much older 1930s-1940s posters that the user considers too old for this plugin instance.
- The approved display direction is a three-vertical-poster triptych over a soft blurred backdrop, with the posters themselves pasted plainly: no frame, no shadow, no matte card.
- Single landscape poster mode should also be plain contain on a solid background, not a blur-backed framed layout.
