# 二狗新闻早报固定规则

## 输出目标

生成一张竖版中文日报图片，面向手机阅读。每次运行只刷新当天内容，不重新设计版式。

## 版式规则

- 标题固定为 `二狗新闻早报`。
- 副标题固定为 `1分钟知天下事`。
- 使用北京日期。
- 保持报纸/城市背景、红色标题、白色圆角内容卡片、红色编号、金色观察表的视觉方向。
- 画面左右必须留出充足安全边距，主内容区不要贴近画布边缘。
- 每张卡片内部也要留白，避免文字、编号和卡片边框互相挤压。
- 竖版手机图优先使用单列主阅读流；底部小模块可以并排，但必须留出明显间隔。
- 如果内容很多，优先缩短每条文字和增加行距，不要把文字压到两侧。
- 不做营销海报风，不做纯文字文档，不随意改变版式结构。

## 内容规则

- `今日要闻` 要具体，不要泛泛宏观概括。每条必须有短标题和事实详情，详情里优先包含时间、地点/主体、数字、动作、结果或影响。
- `今日要闻` 不接受“多地发布”“部门通报”“市场关注”这类空泛写法，除非同一句给出具体平台数、城市数、人数、指数、涨跌幅、响应级别、执行日期等可核查数字。
- `今日意外` 固定 8 条，使用国内已验证事故、突发、官方通报或主流媒体更新。每条必须写明发生或更新的时间/地区、事件类型、处置状态和可核查数字。
- `今日意外` 从新到旧排序，按发生时间或最新官方更新时间排序。
- 如果当天同类事件不足，用最近几天的最新官方后续，不用未经证实的小道消息凑数。
- `洛阳天气` 默认使用河南洛阳，摄氏度。
- `市场快照` 包含 A 股和美股。如果当天休市或尚未收盘，要明确标注使用最近可用收盘或休市背景。
- `三只买入观察` 是观察清单，不构成投资建议。
- 页脚保留来源/时间说明。
