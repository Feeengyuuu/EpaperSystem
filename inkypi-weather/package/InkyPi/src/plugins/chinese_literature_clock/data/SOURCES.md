# Chinese Literature Clock Sources

The bundled quote seed data is extracted from public-domain Project Gutenberg
Chinese-language ebooks. The extractor keeps the source text as published,
strips Gutenberg boilerplate, removes hard line wrapping, and selects sentences
that contain either precise clock expressions or traditional Chinese time
periods.

Project Gutenberg ebook pages are available at
`https://www.gutenberg.org/ebooks/<id>`.
