Use the Blizzard slug, for example area-52, illidan, stormrage.
Public character mode uses BLIZZARD_CLIENT_ID and BLIZZARD_CLIENT_SECRET from API Keys. Existing WoW_Key is accepted as a client-id alias, but Blizzard still requires a separate client secret. Account-wide mode requires a user OAuth token with wow.profile scope as BLIZZARD_USER_ACCESS_TOKEN.
