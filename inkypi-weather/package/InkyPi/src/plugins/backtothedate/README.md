# BacktotheDate

BacktotheDate fetches one random poster image from `https://chineseposters.net/`
each time the plugin is displayed in a playlist.

The plugin uses a horizontal-display mode by default: each poster stays
complete, and any surrounding space is filled with a blurred version of the
same image. Portrait posters rotate 90 degrees counterclockwise before
rendering so they fit the horizontal e-paper display more naturally. Use `Show
full poster on plain background` for a plain mat, or `Fill screen by cropping`
only if edge cropping is acceptable.

Source and copyright guidance are provided by Chineseposters.net:
https://chineseposters.net/about/faqs
